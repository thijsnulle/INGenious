# YAML-Based Object Repository Implementation Plan

## Executive Summary

This plan outlines the implementation of YAML-based Object Repositories for INGenious, replacing the verbose XML format with a clean, human-readable YAML structure that supports page-per-file organization for better version control.

## Current State Analysis

### Existing OR Structure

```
ObjectRepository
├── WebOR (OR.object - XML file)
│   └── WebORPage[]
│       └── ObjectGroup<WebORObject>[]
│           └── WebORObject[]
│               └── ORAttribute[] (10 properties)
├── MobileOR (MOR.object - XML file)  
└── ImageOR (IOR.object - XML file)
```

### Current XML Format (OR.object)

```xml
<Root ref="ProjectName" type="OR">
  <Page ref="ContactUs">
    <ObjectGroup ref="firstName">
      <Object ref="firstName" frame="">
        <Property ref="Role" value="" pref="1"/>
        <Property ref="Text" value="" pref="2"/>
        <Property ref="Label" value="First name *(required)" pref="3"/>
        <Property ref="Placeholder" value="" pref="4"/>
        <Property ref="xpath" value="" pref="5"/>
        <Property ref="css" value="" pref="6"/>
        <Property ref="AltText" value="" pref="7"/>
        <Property ref="Title" value="" pref="8"/>
        <Property ref="TestId" value="" pref="9"/>
        <Property ref="ChainedLocator" value="" pref="10"/>
      </Object>
    </ObjectGroup>
  </Page>
</Root>
```

**Problems with Current Format:**
- ❌ All 10 properties stored even when empty
- ❌ Verbose XML syntax (145 lines for 10 elements)
- ❌ Single file = merge conflicts in Git
- ❌ Hard to read/edit manually
- ❌ No clear locator priority indication

## Proposed YAML Format

### Option A: Page-Per-File (Recommended)

**Directory Structure:**
```
ProjectName/
├── OR.object              # Legacy XML (kept for backward compatibility)
├── pages/                 # NEW: YAML page files
│   ├── _index.yaml        # Page metadata and order
│   ├── home.yaml
│   ├── contact-us.yaml
│   ├── login/             # Nested pages for complex sites
│   │   ├── login-form.yaml
│   │   └── forgot-password.yaml
│   └── checkout/
│       ├── cart.yaml
│       └── payment.yaml
└── ObjectRepository/      # Existing CSV backup files
```

### YAML Schema Design

#### Page File (pages/contact-us.yaml)

```yaml
# Contact Us Page - Object Repository
# Auto-generated by INGenious v2.3.1

page: ContactUs
description: Contact form page with inquiry submission
url_pattern: /contact-us
tags: [forms, contact, public]

elements:
  # Form fields - using semantic locators
  firstName:
    label: "First name *(required)"
    description: First name input field
    
  lastName:
    label: "Last name *(required)"
    
  email:
    label: "E-mail address *(required)"
    placeholder: "example@domain.com"
    
  phone:
    label: "Phone number"
    testId: contact-phone
    
  message:
    label: "Your message *(required)"
    role: textbox
    
  # Buttons
  submitButton:
    role: button
    text: Send
    description: Submit contact form
    
  cancelButton:
    role: button
    text: Cancel
    css: "button.cancel-btn"  # Fallback CSS selector
    
  # Complex element with multiple locators (priority order)
  searchBox:
    testId: search-input          # Priority 1 (most stable)
    placeholder: "Search..."       # Priority 2
    css: "input.search-field"     # Priority 3 (fallback)
    frame: header-frame           # Optional frame context
    
  # Dynamic element with XPath
  dynamicRow:
    xpath: "//tr[contains(@class, 'dynamic-row')][{index}]"
    description: Dynamic table row (use with data substitution)
    
  # Chained locator for complex DOM
  nestedElement:
    chainedLocator: "parentDiv >> childSpan >> targetInput"
    description: Element nested inside shadow DOM
```

#### Index File (pages/_index.yaml)

```yaml
# Object Repository Index
# Defines page order and metadata

version: "1.0"
format: yaml
generated: 2024-01-15T10:30:00Z
lastModified: 2024-01-15T14:22:00Z

pages:
  - name: HomePage
    file: home.yaml
    tags: [navigation, public]
    
  - name: ContactUs
    file: contact-us.yaml
    tags: [forms, public]
    
  - name: Login
    folder: login/
    children:
      - name: LoginForm
        file: login-form.yaml
      - name: ForgotPassword
        file: forgot-password.yaml
        
settings:
  defaultLocatorPriority:
    - testId      # Most stable
    - role
    - label
    - text
    - placeholder
    - css
    - xpath       # Least stable
    - chainedLocator
```

#### YamlPageDefinition.java

```java
package com.ing.datalib.or.yaml;

import com.fasterxml.jackson.annotation.JsonInclude;
import com.fasterxml.jackson.annotation.JsonPropertyOrder;
import java.util.LinkedHashMap;
import java.util.List;
import java.util.Map;

@JsonInclude(JsonInclude.Include.NON_EMPTY)
@JsonPropertyOrder({"page", "description", "url_pattern", "tags", "elements"})
public class YamlPageDefinition {
    
    private String page;
    private String description;
    private String urlPattern;
    private List<String> tags;
    private Map<String, YamlElementDefinition> elements = new LinkedHashMap<>();
    
    // Getters and setters...
    
    public static YamlPageDefinition fromWebORPage(WebORPage page) {
        YamlPageDefinition yaml = new YamlPageDefinition();
        yaml.setPage(page.getName());
        
        for (ObjectGroup<WebORObject> group : page.getObjectGroups()) {
            for (WebORObject obj : group.getObjects()) {
                YamlElementDefinition element = YamlElementDefinition.fromWebORObject(obj);
                yaml.getElements().put(obj.getName(), element);
            }
        }
        return yaml;
    }
    
    public WebORPage toWebORPage() {
        WebORPage page = new WebORPage(this.page);
        for (Map.Entry<String, YamlElementDefinition> entry : elements.entrySet()) {
            ObjectGroup<WebORObject> group = new ObjectGroup<>(entry.getKey(), page);
            WebORObject obj = entry.getValue().toWebORObject(entry.getKey(), group);
            group.addObject(obj);
            page.addObjectGroup(group);
        }
        return page;
    }
}
```

#### YamlElementDefinition.java

```java
package com.ing.datalib.or.yaml;

import com.fasterxml.jackson.annotation.JsonAnyGetter;
import com.fasterxml.jackson.annotation.JsonAnySetter;
import com.fasterxml.jackson.annotation.JsonIgnore;
import com.fasterxml.jackson.annotation.JsonInclude;
import java.util.LinkedHashMap;
import java.util.Map;

@JsonInclude(JsonInclude.Include.NON_EMPTY)
public class YamlElementDefinition {
    
    // Standard locator properties (use lowercase for YAML)
    private String role;
    private String text;
    private String label;
    private String placeholder;
    private String xpath;
    private String css;
    private String altText;
    private String title;
    private String testId;
    private String chainedLocator;
    
    // Additional metadata
    private String frame;
    private String description;
    
    // Capture any unknown properties for extensibility
    @JsonIgnore
    private Map<String, Object> additionalProperties = new LinkedHashMap<>();
    
    @JsonAnyGetter
    public Map<String, Object> getAdditionalProperties() {
        return additionalProperties;
    }
    
    @JsonAnySetter
    public void setAdditionalProperty(String key, Object value) {
        additionalProperties.put(key, value);
    }
    
    public static YamlElementDefinition fromWebORObject(WebORObject obj) {
        YamlElementDefinition elem = new YamlElementDefinition();
        elem.setFrame(obj.getFrame());
        
        for (ORAttribute attr : obj.getAttributes()) {
            if (attr.getValue() != null && !attr.getValue().isEmpty()) {
                switch (attr.getName().toLowerCase()) {
                    case "role": elem.setRole(attr.getValue()); break;
                    case "text": elem.setText(attr.getValue()); break;
                    case "label": elem.setLabel(attr.getValue()); break;
                    case "placeholder": elem.setPlaceholder(attr.getValue()); break;
                    case "xpath": elem.setXpath(attr.getValue()); break;
                    case "css": elem.setCss(attr.getValue()); break;
                    case "alttext": elem.setAltText(attr.getValue()); break;
                    case "title": elem.setTitle(attr.getValue()); break;
                    case "testid": elem.setTestId(attr.getValue()); break;
                    case "chainedlocator": elem.setChainedLocator(attr.getValue()); break;
                }
            }
        }
        return elem;
    }
    
    public WebORObject toWebORObject(String name, ObjectGroup group) {
        WebORObject obj = new WebORObject(name, group);
        
        // Set attribute values (only non-empty ones matter in YAML)
        setAttributeIfPresent(obj, "Role", role);
        setAttributeIfPresent(obj, "Text", text);
        setAttributeIfPresent(obj, "Label", label);
        setAttributeIfPresent(obj, "Placeholder", placeholder);
        setAttributeIfPresent(obj, "xpath", xpath);
        setAttributeIfPresent(obj, "css", css);
        setAttributeIfPresent(obj, "AltText", altText);
        setAttributeIfPresent(obj, "Title", title);
        setAttributeIfPresent(obj, "TestId", testId);
        setAttributeIfPresent(obj, "ChainedLocator", chainedLocator);
        
        if (frame != null && !frame.isEmpty()) {
            obj.setFrame(frame);
        }
        
        return obj;
    }
    
    private void setAttributeIfPresent(WebORObject obj, String name, String value) {
        if (value != null && !value.isEmpty()) {
            obj.getAttributeByName(name).setValue(value);
        }
    }
}
```

### Phase 2: ObjectRepository Enhancement

#### Modified ObjectRepository.java

```java
package com.ing.datalib.or;

import com.ing.datalib.or.yaml.*;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.dataformat.yaml.YAMLFactory;
import com.fasterxml.jackson.dataformat.yaml.YAMLGenerator;
import java.io.File;
import java.nio.file.*;

public class ObjectRepository {
    
    private static final XmlMapper XML_MAPPER = new XmlMapper();
    private static final ObjectMapper YAML_MAPPER = createYamlMapper();
    
    private ORFormat preferredFormat = ORFormat.YAML;  // Default to YAML
    
    public enum ORFormat { XML, YAML }
    
    private static ObjectMapper createYamlMapper() {
        YAMLFactory factory = new YAMLFactory()
            .disable(YAMLGenerator.Feature.WRITE_DOC_START_MARKER)
            .enable(YAMLGenerator.Feature.MINIMIZE_QUOTES);
        return new ObjectMapper(factory);
    }
    
    private void init() {
        // Try YAML first, fall back to XML
        File pagesDir = new File(getPagesLocation());
        
        if (pagesDir.exists() && hasYamlPages(pagesDir)) {
            webOR = loadFromYaml();
            preferredFormat = ORFormat.YAML;
        } else if (new File(getORLocation()).exists()) {
            webOR = loadFromXml();
            // Auto-migrate: convert to YAML on first load
            if (shouldAutoMigrate()) {
                saveAsYaml();
            }
        } else {
            webOR = new WebOR(sProject.getName());
        }
    }
    
    private WebOR loadFromYaml() {
        WebOR or = new WebOR(sProject.getName());
        File pagesDir = new File(getPagesLocation());
        
        for (File yamlFile : pagesDir.listFiles((d, n) -> n.endsWith(".yaml"))) {
            if (yamlFile.getName().equals("_index.yaml")) continue;
            
            try {
                YamlPageDefinition pageDef = YAML_MAPPER.readValue(
                    yamlFile, YamlPageDefinition.class);
                WebORPage page = pageDef.toWebORPage();
                page.setRoot(or);
                or.getPages().add(page);
            } catch (IOException e) {
                Logger.getLogger(ObjectRepository.class.getName())
                    .log(Level.WARNING, "Failed to load: " + yamlFile, e);
            }
        }
        return or;
    }
    
    public void saveAsYaml() {
        File pagesDir = new File(getPagesLocation());
        pagesDir.mkdirs();
        
        for (WebORPage page : webOR.getPages()) {
            YamlPageDefinition pageDef = YamlPageDefinition.fromWebORPage(page);
            String fileName = toKebabCase(page.getName()) + ".yaml";
            
            try {
                File yamlFile = new File(pagesDir, fileName);
                YAML_MAPPER.writerWithDefaultPrettyPrinter()
                    .writeValue(yamlFile, pageDef);
            } catch (IOException e) {
                Logger.getLogger(ObjectRepository.class.getName())
                    .log(Level.SEVERE, "Failed to save: " + fileName, e);
            }
        }
        
        // Write index file
        writeIndexFile(pagesDir);
    }
    
    public String getPagesLocation() {
        return sProject.getLocation() + File.separator + "pages";
    }
    
    private String toKebabCase(String name) {
        return name.replaceAll("([a-z])([A-Z])", "$1-$2")
                   .replaceAll("\\s+", "-")
                   .toLowerCase();
    }
}
```

## Migration Strategy

### Backward Compatibility

1. **Dual Format Support**: Read XML or YAML based on what exists
2. **Auto-Migration**: Convert XML to YAML on first load (configurable)
3. **Fallback**: If YAML parsing fails, try XML
4. **Legacy Export**: Always maintain ability to export to XML

### Migration Command

```bash
# Convert existing XML to YAML
./ingenious or convert --from xml --to yaml --project ./Projects/MyProject

# Validate YAML files
./ingenious or validate ./Projects/MyProject/pages/

# Import external YAML OR
./ingenious or import --input ./external-or/ --project ./Projects/MyProject
```
